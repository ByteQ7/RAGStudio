# IT 服务台支持手册

## 一、服务范围

IT 服务台为全体员工提供以下技术支持：

| 服务类别 | 服务内容 | 响应时间 |
|---------|---------|---------|
| 账号管理 | 新建/禁用/重置密码 | 4 小时内 |
| 网络故障 | 有线/无线网络连接问题 | 2 小时内 |
| 硬件故障 | 电脑、打印机、显示器等设备故障 | 8 小时内 |
| 软件安装 | 办公软件、开发工具安装配置 | 8 小时内 |
| 权限申请 | 系统权限、文件夹访问权限 | 24 小时内 |

## 二、报修方式

1. **IT 服务台热线**：分机 8888
2. **IT 服务台邮箱**：it-support@company.com
3. **OA 系统**：登录 OA → IT 服务 → 提交工单
4. **企业微信**：搜索「IT 服务台」小程序

## 三、常用操作指南

### 3.1 密码重置
- **OA 系统密码**：登录页点击"忘记密码"，通过绑定手机或邮箱重置。
- **邮箱密码**：联系 IT 服务台，验证身份后重置。
- **Wi-Fi 密码**：连接公司 Wi-Fi（SSID: Company-WiFi），输入工号+默认密码 123456 登录。

### 3.2 VPN 连接
- 下载并安装公司 VPN 客户端（下载地址：https://vpn.company.com/download）。
- 使用工号和 OA 密码登录。
- VPN 地址：vpn.company.com，端口：443。

### 3.3 打印机设置
- 驱动下载：\\print-server\drivers。
- 手动添加打印机：控制面板 → 设备和打印机 → 添加打印机 → 选择对应型号。
- 默认打印机：各个楼层公共区域的 Printer-Floor1~Printer-Floor6。

## 四、设备领用与归还

### 领用流程
1. 新员工入职当天，IT 服务台配发办公设备。
2. 标准配置：笔记本电脑一台、显示器一台、键盘鼠标一套、耳机一副。
3. 领用设备时需签字确认《设备领用单》。

### 归还流程
1. 员工离职前需将全部设备归还 IT 服务台。
2. 归还前请备份个人数据，IT 服务台将对设备进行数据清除。
3. 设备丢失或人为损坏需按设备原价赔偿。

## 五、信息安全注意事项

1. 办公电脑必须安装公司指定的杀毒软件。
2. 不得在办公电脑上安装未经授权的软件。
3. 公司邮箱仅限工作使用，不得用于注册个人账号。
4. 发现异常登录或可疑邮件，请立即报告 IT 服务台。
5. 离开工位时必须锁定电脑（Win+L）。
