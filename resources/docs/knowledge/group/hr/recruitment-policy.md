# 员工招聘与入职管理制度

## 一、招聘流程

```
用人部门提出需求 → 人力资源部审核 → 审批 → 发布招聘信息 → 简历筛选 → 面试 → 录用审批 → 发放 Offer → 入职
```

## 二、面试流程

### 面试轮次
- **普通员工**：HR 初试 → 部门负责人复试 → 分管领导终试。
- **主管/经理级**：HR 初试 → 部门负责人复试 → 分管领导终试 → 总经理终试。
- **总监及以上**：HR 初试 → 分管领导复试 → 总经理终试 → 董事长终试。

### 面试评估
- 面试官需填写《面试评估表》，从专业能力、沟通能力、团队协作、发展潜力等维度评分。
- 面试结论分为：强烈推荐、推荐、待定、不推荐。

## 三、入职流程

1. 收到 Offer 后 3 个工作日内确认入职意向。
2. 入职当天携带以下资料到人力资源部办理手续：
   - 身份证原件及复印件
   - 学历学位证书原件及复印件
   - 离职证明（上家公司开具）
   - 近期一寸免冠照片 2 张
   - 银行卡复印件（工资卡）
   - 体检报告（近 3 个月内）
3. 签订劳动合同（首次合同期限 3 年，试用期 6 个月）。
4. 领取办公设备，IT 部门开通账号权限。
5. 部门负责人安排入职引导人，进行岗位交接。

## 四、试用期管理

- 试用期 6 个月，试用期工资为转正工资的 80%。
- 试用期内参加新员工培训和入职考核。
- 试用期考核不合格者，公司可解除劳动合同。
- 试用期表现优异者可申请提前转正（满 3 个月后可申请）。

## 五、离职管理

- **试用期离职**：提前 3 天提出。
- **正式员工离职**：提前 30 天提交书面《离职申请单》。
- 离职流程：部门审批 → 工作交接 → IT 设备归还 → 财务结算 → 人力资源部办理离职手续。
- 离职结算：工资结算至离职当日，未休年假按日工资 300% 补偿。
