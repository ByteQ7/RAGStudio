# 薪酬福利管理制度

## 一、薪酬结构

员工薪酬 = 基本工资 + 岗位工资 + 绩效工资 + 补贴 + 年终奖

| 构成部分 | 说明 | 发放方式 |
|---------|------|---------|
| 基本工资 | 根据岗位等级确定 | 月发 |
| 岗位工资 | 根据岗位职责和技能要求确定 | 月发 |
| 绩效工资 | 根据季度考核结果浮动 | 季度发放 |
| 补贴 | 餐补、交通补、通讯补等 | 月发 |
| 年终奖 | 根据年度考核结果发放 | 次年春节前发放 |

## 二、调薪机制

### 年度调薪
每年 4 月根据年度绩效考核结果进行调薪：
- S 级：上调 15%~20%
- A 级：上调 8%~12%
- B 级：上调 3%~5%
- C 级：不调薪
- D 级：降薪或淘汰

### 晋升调薪
岗位晋升后，薪酬调整至新岗位的薪酬带宽内，原则上涨幅不低于 15%。

## 三、福利体系

### 法定福利
- 五险一金（养老保险、医疗保险、失业保险、工伤保险、生育保险、住房公积金）。
- 住房公积金缴纳比例：公司 12%、个人 12%。

### 补充福利
- **补充医疗保险**：员工及子女可享受补充商业医疗保险。
- **年度体检**：每年一次免费健康体检。
- **节日福利**：春节、端午、中秋等传统节日发放节日礼品或礼金（标准 500 元/次）。
- **生日福利**：员工生日当月发放生日礼金 300 元。
- **团建活动**：每季度部门团建经费 200 元/人。
- **零食下午茶**：每周五下午提供水果和零食。
