# 员工奖惩管理条例

## 一、奖励

### 奖励类别

| 奖励类别 | 适用情形 | 奖励标准 |
|---------|---------|---------|
| 通报表扬 | 工作表现突出、获得客户表扬 | 全公司通报表扬 |
| 月度之星 | 月度绩效考核 S 级 | 奖金 2000 元 |
| 技术创新奖 | 提出技术改进方案并落地实施 | 奖金 5000~20000 元 |
| 优秀团队奖 | 团队业绩突出或完成重大项目 | 团队奖金 50000 元 |
| 长期服务奖 | 入职满 5 年/10 年/15 年/20 年 | 纪念奖牌 + 奖金 |

### 奖励流程
```
部门推荐/个人自荐 → 人力资源部审核 → 评审委员会评定 → 公示 → 发放
```

## 二、违纪处分

### 处分等级

| 处分等级 | 适用情形 | 处罚措施 |
|---------|---------|---------|
| 警告 | 轻微违纪、首次违规 | 书面警告，通报部门 |
| 记过 | 较严重违纪或再次违规 | 记过处分，绩效降级 |
| 降级 | 严重违纪或多次违规 | 降职降薪 |
| 解除合同 | 重大违纪 | 辞退，不支付补偿 |

### 重大违纪行为
1. 严重违反公司制度，给公司造成重大损失（损失金额 50000 元以上）。
2. 利用职务之便收受贿赂或回扣。
3. 泄露公司商业秘密。
4. 虚构报销、骗取公司资金。
5. 打架斗殴、性骚扰等严重违反公序良俗的行为。
6. 连续旷工 3 天或年度累计旷工 5 天。

## 三、申诉

员工对处分决定有异议的，可在收到处分通知后 7 个工作日内向人力资源部提出申诉，申诉委员会在 15 个工作日内作出裁决。
