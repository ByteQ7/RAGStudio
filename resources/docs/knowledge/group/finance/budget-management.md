# 财务预算管理制度

## 一、预算编制

### 编制时间
- 年度预算：每年 10 月启动下一年度预算编制。
- 季度调整：每季度末可对下一季度预算进行调整。

### 编制流程
```
预算通知下发 → 各部门编制预算 → 财务部汇总初审 → 预算委员会评审 → 总经理审批 → 下达执行
```

### 预算科目
- **收入预算**：主营业务收入、其他业务收入。
- **成本预算**：原材料成本、人力成本、运营成本。
- **费用预算**：销售费用、管理费用、研发费用、财务费用。
- **资本预算**：固定资产购置、信息化建设、投资并购。

## 二、预算执行

- 各部门需按月编制预算执行报告，偏差超过 10% 需说明原因。
- 预算内支出按正常审批流程执行。
- 超预算支出需提交《预算追加申请》，说明追加原因和资金来源。

## 三、预算审批权限

| 金额范围 | 审批人 |
|---------|--------|
| 5000 元以下 | 部门负责人 |
| 5000~50000 元 | 分管领导 |
| 50000~200000 元 | 总经理 |
| 200000 元以上 | 董事会 |

## 四、预算考核

- 预算执行准确率纳入部门年度绩效考核。
- 偏差率在 ±5% 以内为优秀，±10% 以内为合格。
- 偏差率超过 ±20% 需提交专项说明。
