# 印章管理制度

## 一、总则

为加强公司印章管理，规范印章使用行为，维护公司合法权益，防范经营风险，根据国家有关法律法规及公司实际情况，特制定本制度。

本制度适用于公司及所属分支机构的各类印章，包括公章、合同专用章、财务专用章、发票专用章、法定代表人章、部门章以及各类业务专用章的刻制、保管、使用、作废等管理工作。

## 二、印章的种类及用途

### 2.1 公司级印章

| 印章类型 | 保管部门 | 用途 | 刻制数量 |
|---------|---------|------|---------|
| 公章 | 行政部 | 公司对外正式文件、函件、证明等 | 1 枚 |
| 合同专用章 | 法务部 | 各类经济合同的签署 | 1 枚 |
| 财务专用章 | 财务部 | 银行往来业务、财务文件 | 1 枚 |
| 发票专用章 | 财务部 | 发票开具、发票相关业务 | 1 枚 |
| 法定代表人章 | 行政部 | 银行支票、重要文件签署 | 1 枚 |

### 2.2 其他印章

| 印章类型 | 保管部门 | 用途 |
|---------|---------|------|
| 部门章 | 各部门 | 部门内部文件、证明 |
| 投标专用章 | 市场部 | 投标文件 |
| 技术专用章 | 技术部 | 技术文档、图纸 |
| 人力资源部章 | 人力资源部 | 人事文件、证明 |
| 归档章 | 行政部 | 档案归档标识 |

## 三、印章的刻制与启用

### 3.1 刻制审批
1. 新设印章需由需求部门填写《印章刻制申请表》，注明印章名称、规格、用途。
2. 经部门负责人审核后，报分管领导审批。
3. 公司级印章的刻制需经总经理审批。

### 3.2 刻制要求
1. 印章必须到公安机关指定的具有合法资质的印章刻制企业刻制。
2. 印章规格、材质、字体、排列方式等应符合公安机关的规定。
3. 公章和合同专用章应带有防伪编码。
4. 财务专用章、发票专用章的规格应符合银行和税务机关的要求。

### 3.3 启用程序
1. 印章刻制完成后，由行政部进行印模备案。
2. 填写《印章启用登记表》，记录印章名称、刻制日期、启用日期、保管人等信息。
3. 通知公司各部门印章已启用。
4. 新印章启用同时，旧印章应收回并作废处理。

## 四、印章的保管

### 4.1 保管原则
1. 印章实行专人保管、专柜存放、专册登记制度。
2. 印章保管人应为本部门正式员工，由部门负责人指定。
3. 印章保管人离职或岗位变动前，应办理印章交接手续，填写《印章交接登记表》。
4. 印章存放于保险柜或带锁的铁皮柜中，严禁随意摆放。

### 4.2 保管要求
1. 印章应保持清洁，印迹清晰可辨。
2. 印章发现问题（如磨损、变形等），应及时申请更换。
3. 印章保管人应定期检查印章完好情况。
4. 严禁将印章携带出公司使用。确因工作需要外带的，需经总经理审批，并指定两人同行、全程监管。

### 4.3 印章遗失处理
1. 发现印章遗失或被盗，保管人应立即报告部门负责人和行政部。
2. 行政部在 1 小时内向公安机关报案，并登报声明印章作废。
3. 同步通知各业务合作方、银行等关联单位。
4. 查明遗失原因，追究相关责任人责任。
5. 及时申请刻制新印章。

## 五、印章的使用

### 5.1 使用审批

#### 公章

| 文件类型 | 审批流程 |
|---------|---------|
| 常规文件（函件、证明等） | 主办部门负责人 → 行政部负责人 → 用印 |
| 重要文件（合同除外） | 主办部门负责人 → 分管领导 → 行政部负责人 → 用印 |
| 特别重大文件（投融资、担保等） | 主办部门负责人 → 分管领导 → 总经理 → 行政部负责人 → 用印 |

#### 合同专用章
按《合同管理制度》的审批流程执行。经审批通过的合同，凭《合同审批单》到法务部用印。

#### 财务专用章
- 银行结算业务：按公司资金管理规定执行。
- 其他财务文件：财务负责人审批。

#### 发票专用章
- 发票开具时由财务部指定专人使用。
- 非发票类文件不得使用发票专用章。

#### 法定代表人章
- 银行支票：按公司资金管理规定执行。
- 其他文件：法定代表人本人或经其书面授权的人员使用。

### 5.2 用印流程

```
经办人填写《印章使用申请表》→ 部门负责人审核 → 归口审批人审批 → 印章保管人核实用印内容 → 用印 → 登记
```

### 5.3 用印规范
1. 用印前必须核对用印文件内容，确保已经过完整审批流程。
2. 严禁在空白纸张、空白介绍信、空白合同等空白文件上盖印。
3. 用印位置应准确：公章盖在文件落款处，上不压正文，下要骑年盖月。
4. 盖章应端正、清晰，不得模糊或倾斜。
5. 多页文件需加盖骑缝章，确保各页属于同一份文件。
6. 文件份数较多时，应逐一核对份数后统一盖印。
7. 严禁在不完整的合同上盖印，合同附件需加盖骑缝章。

### 5.4 用印登记
1. 每次用印需在《印章使用登记簿》上登记，内容包括：用印日期、文件名称、用印类型、批准人、经办人、用印份数。
2. 《印章使用登记簿》应妥善保管，保存期限为 5 年。
3. 印章保管人每月对用印记录进行汇总，报部门负责人审阅。

## 六、印章的停用与作废

### 6.1 停用情形
以下情况印章应停用：
1. 公司名称变更、法定代表人变更。
2. 部门撤销或合并。
3. 印章严重磨损、变形无法正常使用。
4. 印章遗失后已登报声明作废。

### 6.2 停用流程
1. 停用印章由保管部门填写《印章停用申请表》，经原审批人批准后停用。
2. 停用的印章应交回行政部统一保管或销毁。
3. 行政部对停用印章进行登记，注明停用日期和原因。

### 6.3 销毁处理
1. 停用的印章应由行政部统一销毁。
2. 印章销毁需两人以上在场监督，并做好销毁记录。
3. 销毁记录应包括：印章名称、刻制日期、停用日期、销毁日期、销毁方式、监销人签名。
4. 销毁记录永久保存。

## 七、印章使用异常处理

### 7.1 印章被盗用或冒用
1. 发现印章被盗用或冒用，保管人应立即报告部门负责人和行政部。
2. 行政部应在 2 小时内向公安机关报案，并取得报案回执。
3. 同步通知各业务合作单位、银行等关联机构，声明印章作废。
4. 通过公司公告或登报方式声明印章作废。
5. 对被盗用或冒用期间产生的法律后果进行评估，法务部根据情况采取应对措施。

### 7.2 印章出现破损或变形
1. 印章因长期使用出现磨损、变形或印迹模糊的，应及时申请更换。
2. 破损印章由行政部收回并销毁。
3. 新印章刻制后，按启用程序办理。
4. 新旧印章交替期间，应做好衔接安排，避免影响业务正常开展。

### 7.3 印章争议处理
如对已盖章文件的内容或效力产生争议，应立即将争议情况报告法务部。法务部负责调查核实印章使用情况，包括查阅用印登记簿、审批单等原始记录，确定印章使用的合规性和文件的真实效力。

## 八、监督检查

### 7.1 定期检查
1. 行政部每季度对各部门印章保管和使用情况进行检查。
2. 检查内容包括：印章存放是否安全、使用登记是否完整、审批流程是否合规。
3. 检查结果形成书面报告，提交分管领导。

### 7.2 违规处罚
1. 未经审批擅自使用印章的，对责任人给予记过或降级处分。
2. 在空白文件上盖印的，按严重违纪处理。给公司造成损失的，责任人承担赔偿责任。
3. 印章保管人擅自将印章交予他人使用的，给予警告或记过处分。
4. 印章遗失未及时报告的，对保管人和部门负责人给予记过或降级处分。
5. 利用印章从事违法犯罪活动的，公司依法追究法律责任。

## 八、附则

1. 本制度由行政部负责解释和修订。
2. 本制度未尽事宜，参照国家相关法律法规及公司其他制度执行。
3. 本制度自发布之日起执行，此前相关规定与本制度不一致的，以本制度为准。

## 九、电子印章管理

### 9.1 电子印章的申请与制作
1. 电子印章的申请流程与实体印章一致，需填写《电子印章申请表》。
2. 电子印章由公司指定的电子认证服务机构制作。
3. 电子印章应与实体印章具有同等的法律效力。
4. 电子印章的图形应与实体印章一致。

### 9.2 电子印章的使用
1. 电子印章的使用审批流程与实体印章相同。
2. 电子印章的使用应在合法的电子合同签署平台上进行。
3. 每次使用电子印章需进行身份认证（密码、短信验证码或指纹识别）。
4. 电子印章使用记录由系统自动保存，包括使用人、使用时间、签署文件等信息。

### 9.3 电子印章的保管
1. 电子印章密钥文件应加密存储，不得通过互联网传输。
2. 电子印章使用设备应设置开机密码和屏幕锁。
3. 电子印章保管人离职或岗位变动时，应办理密钥交接手续，并重置访问权限。

## 十一、常见问题解答（FAQ）

### Q1: 合同可以先盖章后审批吗？
严禁先盖章后审批。所有用印必须经过完整的审批流程，审批通过后方可用印。先盖章后审批的行为一经发现，将追究经办人和保管人的责任。

### Q2: 印章保管人临时不在岗怎么办？
印章保管人因出差、请假等原因不在岗时，应提前办理印章临时交接手续，由部门负责人指定临时保管人。交接双方需在《印章交接登记表》上签字确认。

### Q3: 子公司或分支机构的印章如何管理？
子公司或分支机构的印章由公司统一刻制，印章保管人和使用审批权限由公司授权。印章使用数据应定期向公司行政部报备。

### Q4: 作废的印章如何处理？
作废的印章由行政部统一回收，经公司领导批准后，在监销人监督下销毁。销毁方式为切割破坏或熔毁，确保无法复原。

### Q5: 电子印章的效力是否与实体印章相同？
根据《中华人民共和国电子签名法》，可靠的电子印章与实体印章具有同等的法律效力。电子印章的使用需确保电子认证机构具有合法资质。
