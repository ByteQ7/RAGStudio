# 印章管理制度

## 一、印章种类

| 印章类型 | 保管部门 | 使用范围 |
|---------|---------|---------|
| 公章 | 行政部 | 公司对外重要文件 |
| 合同专用章 | 法务部 | 合同签署 |
| 财务专用章 | 财务部 | 银行往来、财务文件 |
| 发票专用章 | 财务部 | 发票开具 |
| 法定代表人章 | 行政部 | 银行支票、重要文件 |

## 二、印章使用审批

### 公章
- 普通文件：部门负责人 → 行政部负责人 → 用印。
- 重要文件（合同、对外函件）：部门负责人 → 分管领导 → 行政部负责人 → 用印。
- 特别重大文件（投融资、担保等）：需总经理审批。

### 合同专用章
- 三类合同：部门负责人审核 → 法务审核 → 用印。
- 二类合同：部门负责人 → 法务审核 → 分管领导审批 → 用印。
- 一类合同：部门负责人 → 法务审核 → 分管领导 → 总经理审批 → 用印。

## 三、用印规范

1. 用印须填写《印章使用申请表》，注明用印事由、文件名称、份数。
2. 严禁在空白纸张、空白合同上盖印。
3. 盖章清晰、端正，印章不得覆盖正文。
4. 多页文件需加盖骑缝章。

## 四、印章保管

- 印章存放于保险柜中，使用完毕后立即锁存。
- 印章不得带出公司使用，确需外带的需总经理审批并双人陪同。
- 印章遗失或被盗立即报告并启动挂失程序。
