# 合同管理制度

## 一、总则

为规范公司合同管理工作，防范合同风险，维护公司合法权益，促进公司经营活动健康发展，根据《中华人民共和国民法典》及相关法律法规，结合公司实际情况，特制定本制度。

本制度适用于公司对外签订的所有经济合同、协议及其他具有法律约束力的文件（以下统称"合同"）。劳动合同管理按公司人力资源相关制度执行。

## 二、合同管理机构与职责

### 2.1 合同归口管理部门
法务部是公司合同的归口管理部门，主要职责包括：
1. 制定和完善公司合同管理制度。
2. 审核合同的合法性、合规性、完整性。
3. 参与重大合同的谈判和起草工作。
4. 管理合同专用章的使用和保管。
5. 监督合同履行情况，处理合同纠纷。
6. 保管合同档案，建立合同管理台账。
7. 组织开展合同管理培训和普法教育。

### 2.2 合同主办部门
合同主办部门是合同业务的发起和承办部门，主要职责包括：
1. 对合同相对方进行资信调查和尽职调查。
2. 负责合同的商务谈判和条款拟定。
3. 填写《合同审批单》，准备合同附件及相关资料。
4. 按审批流程完成合同审批。
5. 负责合同的履行跟踪和执行反馈。
6. 收集和保管本部门承办的合同资料。

### 2.3 财务部
财务部在合同管理中的主要职责包括：
1. 审核合同中涉及财务条款的内容，包括付款方式、发票类型、税务条款等。
2. 根据合同约定安排付款、开具发票。
3. 对合同涉及的预算额度进行审核确认。
4. 参与重大合同的财务风险评估。

## 三、合同分类与审批权限

### 3.1 合同分类

| 合同类别 | 定义 | 示例 |
|---------|------|------|
| 一类合同 | 标的额 100 万元以上，或涉及公司核心业务、重大资产处置、对外投资、担保等 | 重大采购合同、投资协议、资产转让协议、担保合同 |
| 二类合同 | 标的额 10 万元至 100 万元，或涉及公司重要业务合作 | 普通采购合同、销售合同、技术开发合同、租赁合同 |
| 三类合同 | 标的额 10 万元以下的常规性合同 | 小额采购合同、维修服务合同、保密协议 |

### 3.2 审批权限

#### 一类合同
```
主办部门起草 → 部门负责人审核 → 法务审核 → 财务审核 → 分管领导审批 → 总经理审批 → 法务部盖章归档
```

#### 二类合同
```
主办部门起草 → 部门负责人审核 → 法务审核 → 财务审核 → 分管领导审批 → 法务部盖章归档
```

#### 三类合同
```
主办部门起草 → 部门负责人审核 → 法务审核（可选） → 法务部盖章归档
```

## 四、合同起草与审核

### 4.1 合同起草
1. 优先使用公司制定的标准合同模板。使用非标准合同模板的，需经法务部审核同意。
2. 合同条款应完整、明确、具体，避免模糊不清、产生歧义的表述。
3. 合同中的金额、数量、日期等关键信息需仔细核对，确保准确无误。
4. 合同主体信息应与营业执照等证件一致，包括名称、地址、法定代表人、统一社会信用代码等。

### 4.2 合同必备条款

根据《中华人民共和国民法典》的规定，合同一般应包括以下条款：
1. **当事人的名称或者姓名和住所**：合同双方的基本信息。
2. **标的**：合同指向的对象，如货物、服务、技术等。
3. **数量**：标的的数量、计量单位。
4. **质量**：标的的质量标准、验收标准。
5. **价款或者报酬**：金额、币种、支付方式、支付时间。
6. **履行期限、地点和方式**：交货/服务时间、交付地点、履行方式。
7. **违约责任**：违约金比例、赔偿范围、免责条款。
8. **解决争议的方法**：协商、调解、仲裁或诉讼，以及管辖地。

### 4.3 合同审核要点

#### 法务审核
1. 合同主体是否适格，是否具备相应的资质和履约能力。
2. 合同条款是否违反法律法规强制性规定。
3. 权利义务条款是否对等，是否存在显失公平的条款。
4. 违约责任条款是否合理，违约金比例是否符合法律规定。
5. 争议解决条款是否明确，对公司是否有利。
6. 知识产权归属、保密义务、竞业限制等特殊条款的合理性。

#### 财务审核
1. 付款条款是否符合公司财务制度。
2. 付款方式、付款时间、付款条件是否明确可行。
3. 发票类型（增值税专用发票或普通发票）及开票时间。
4. 合同金额是否在预算范围内。
5. 汇率风险、税务风险等财务风险评估。

## 五、合同签署

### 5.1 签署权限
1. 一类合同由法定代表人或其授权的代理人签署。
2. 二类合同由分管领导或经授权的部门负责人签署。
3. 三类合同由部门负责人签署。

### 5.2 签署规范
1. 合同须加盖公司公章或合同专用章方可生效。
2. 严禁在空白合同或未填写完整的合同文本上盖章。
3. 合同签署页需由授权代表签字，注明签署日期。
4. 合同一式多份的，各份文本内容应完全一致。如有不一致，以加盖公章的文本为准。
5. 合同附件的效力与合同正本相同，附件需加盖骑缝章。

### 5.3 电子合同
1. 经公司批准的电子合同平台签署的电子合同具有同等法律效力。
2. 电子合同需使用合法的电子签名技术，确保签署身份的真实性。
3. 电子合同签署完成后应及时下载存档。

## 六、合同履行

### 6.1 履行跟踪
1. 合同签署后，主办部门应建立合同履行台账，跟踪合同履行进度。
2. 定期检查合同履行情况，及时发现和解决履行中的问题。
3. 对方不履行或履行不符合约定的，应及时书面催告并保留证据。

### 6.2 合同变更
1. 合同变更需签订补充协议，补充协议按原合同的审批权限执行。
2. 合同变更涉及价格、数量、期限等重要条款的，需重新进行法务审核和财务审核。
3. 口头变更无效，必须以书面形式确认。

### 6.3 合同解除
1. 合同解除应签订《合同解除协议》，明确解除原因、责任划分、善后处理等事项。
2. 单方解除合同的，应按照合同约定或法律规定履行通知义务。
3. 合同解除后，主办部门应在 3 个工作日内通知财务部停止付款。

### 6.4 合同归档
1. 合同签署完成后，主办部门应在 5 个工作日内将合同正本及相关资料移交法务部归档。
2. 归档资料包括：合同正本、合同审批单、对方主体资格文件、授权委托书、补充协议、变更协议等。
3. 法务部建立合同管理台账，登记合同编号、合同名称、对方主体、标的额、签署日期、有效期等信息。
4. 合同档案保管期限：合同履行完毕后至少保存 5 年。涉及重大资产、知识产权的合同永久保存。

## 七、合同纠纷处理

### 7.1 纠纷预警
1. 发现对方有违约迹象时，主办部门应立即书面通知法务部。
2. 法务部评估违约风险等级，提出应对方案。
3. 可能引发诉讼或仲裁的纠纷，应在 24 小时内报告总经理。

### 7.2 纠纷处理方式
1. **协商**：优先通过友好协商解决合同纠纷。
2. **调解**：协商不成可申请第三方调解。
3. **仲裁**：合同约定仲裁的，提交约定的仲裁机构仲裁。
4. **诉讼**：合同未约定仲裁或仲裁裁决不服的，可向人民法院提起诉讼。

### 7.3 证据保全
1. 合同履行过程中形成的所有书面文件、电子邮件、即时通讯记录、录音录像等均可能成为诉讼证据，应妥善保管。
2. 发现可能发生纠纷时，应立即对相关证据进行保全。
3. 重要证据可申请公证机关进行证据保全。

## 八、涉外合同管理

### 8.1 适用法律选择
涉外合同应明确约定适用的法律。选择中国法律作为适用法律对公司最为有利。如对方要求适用其他法律，应在合同中明确约定，并请外部律师对法律风险进行评估。

### 8.2 语言版本
涉外合同应同时使用中文和英文（或其他外语）两个版本。约定中英文版本具有同等法律效力，如发生歧义以中文版本为准。翻译工作由公司指派的翻译人员或专业翻译机构完成。

### 8.3 货币与汇率
涉外合同的价款应明确约定计价货币和支付货币。约定汇率风险承担方式，如锁定汇率或浮动汇率。涉及跨境支付的，应遵守国家外汇管理规定，办理外汇登记和备案手续。

### 8.4 争议解决
涉外合同争议解决方式优先选择在中国国际经济贸易仲裁委员会（CIETAC）仲裁。如对方不同意在中国仲裁，可选择在第三国或地区的仲裁机构进行仲裁，如新加坡国际仲裁中心（SIAC）或香港国际仲裁中心（HKIAC）。

## 九、合同印章管理

参照公司《印章管理制度》执行。

## 十、监督检查

### 9.1 定期检查
1. 法务部每季度对合同管理情况进行检查，包括合同审批流程是否合规、合同归档是否及时、合同履行是否存在异常等。
2. 检查结果形成书面报告，提交分管领导和总经理。

### 9.2 违规责任
1. 未按本制度规定履行合同审批程序的，对责任人给予警告或记过处分。
2. 未经授权擅自签署合同的，对责任人给予记过或降级处分。给公司造成损失的，责任人承担相应赔偿责任。
3. 合同档案丢失、损毁的，对责任人给予警告或记过处分。
4. 泄露合同保密信息的，按《保密协议》及相关制度追究责任。

## 十、附则

1. 本制度由法务部负责解释和修订。
2. 本制度未尽事宜，参照国家相关法律法规及公司其他制度执行。
3. 本制度自发布之日起执行，此前相关规定与本制度不一致的，以本制度为准。

## 十一、常见问题解答（FAQ）

### Q1: 合同已经签署但发现条款有误如何更正？
合同签署后发现条款有误，应签订《合同更正协议》或《补充协议》进行更正。重要条款更正需重新履行审批程序。严禁私自涂改已签署的合同文本。

### Q2: 合同到期后未续签但仍在履行怎么办？
合同到期后未续签但双方继续履行的，可能形成事实合同关系，存在法律风险。应在发现后立即启动续签程序，合同到期前 30 天系统应自动提醒。

### Q3: 对方使用电子签名是否具有法律效力？
根据《中华人民共和国电子签名法》，可靠的电子签名与手写签名或者盖章具有同等的法律效力。但涉及婚姻、收养、继承等人身关系的合同，以及法律规定不得使用电子文书的合同除外。

### Q4: 合同争议解决方式选择仲裁还是诉讼？
仲裁一裁终局，效率较高，适合技术性较强的合同纠纷。诉讼两审终审，程序更完善，适合复杂的大额合同纠纷。建议根据合同性质和金额大小选择合适的争议解决方式。

### Q5: 合同的保密期限如何约定？
保密期限一般约定为合同终止后 2-3 年，涉密等级较高的合同可延长至 5 年或更长。涉及商业秘密的，保密期限可约定为永久保密。保密期限应根据信息的重要性和保密需求合理确定。
