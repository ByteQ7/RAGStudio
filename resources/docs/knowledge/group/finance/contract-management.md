# 合同管理制度

## 一、合同分类

| 类别 | 定义 | 审批层级 |
|------|------|---------|
| 一类合同 | 标的额 100 万元以上或涉及公司核心业务的合同 | 总经理审批 |
| 二类合同 | 标的额 10~100 万元的合同 | 分管领导审批 |
| 三类合同 | 标的额 10 万元以下的合同 | 部门负责人审批 |

## 二、合同审批流程

```
合同起草 → 部门审核 → 法务审核 → 财务审核 → 审批 → 盖章 → 归档
```

### 审核要点
- **部门审核**：业务条款的完整性和准确性。
- **法务审核**：法律风险、违约责任、争议解决条款。
- **财务审核**：付款条件、发票类型、税务合规。

## 三、合同签署规范

1. 合同须使用公司统一模板，修改模板需经法务审核。
2. 合同须加盖公司公章或合同专用章方可生效。
3. 严禁在空白合同或未填写完整的合同上盖章。
4. 合同签署页需由授权代表签字，注明签署日期。

## 四、合同管理

- 合同签署后 3 个工作日内归档至合同管理系统。
- 纸质合同原件由行政部统一保管，借阅需登记。
- 合同到期前 30 天，系统自动提醒续签或终止。
- 合同档案保存期限：合同履行完毕后至少保存 5 年。

## 五、违约责任

- 我方违约：按合同约定承担违约责任，同时追究经办部门责任。
- 对方违约：法务部牵头启动违约追偿程序，业务部门配合。
