# 销售管理制度

## 一、销售流程

```
客户开发 → 需求沟通 → 方案制定 → 报价 → 商务谈判 → 合同签订 → 交付执行 → 售后服务
```

## 二、报价管理

### 报价权限
| 折扣范围 | 审批人 |
|---------|--------|
| 标准价 | 销售经理 |
| 标准价 95%~100% | 销售总监 |
| 标准价 85%~95% | 分管副总 |
| 标准价 85% 以下 | 总经理 |

### 报价有效期
- 标准报价有效期为 15 个工作日。
- 超过有效期需重新报价。

## 三、客户管理

### 客户分级
| 级别 | 年度采购额 | 服务策略 |
|------|-----------|---------|
| S 级 | 500 万以上 | 专属客户经理，月度回访 |
| A 级 | 100~500 万 | 客户经理跟进，季度回访 |
| B 级 | 30~100 万 | 销售代表跟进，半年度回访 |
| C 级 | 30 万以下 | 常规跟进 |

### 客户保护
- 已建立合作关系的客户，系统自动分配给对应销售维护。
- 无主客户遵循"首报首跟"原则，以 CRM 系统记录为准。

## 四、回款管理

- 标准账期：合同签订后预付 30%，验收后付 70%。
- 超期 30 天未回款：暂停新项目交付。
- 超期 60 天未回款：移交法务部催收。
- 回款率纳入销售绩效考核，权重 50%。
