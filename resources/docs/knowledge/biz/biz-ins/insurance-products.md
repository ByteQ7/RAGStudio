# 保险业务产品手册

## 一、产品概览

### 1.1 企业财产保险
- **保障范围**：因火灾、爆炸、雷击、暴风、洪水等自然灾害或意外事故造成的财产损失。
- **适用对象**：各类企事业单位。
- **保险期限**：1 年（可续保）。
- **保额上限**：根据投保财产实际价值确定。

### 1.2 公众责任保险
- **保障范围**：因经营活动中发生意外事故，造成第三方人身伤亡或财产损失，依法应承担的赔偿责任。
- **适用对象**：商场、酒店、餐饮、文体场馆等。
- **保险期限**：1 年。
- **每次事故赔偿限额**：100 万~500 万（可选）。

### 1.3 团体意外伤害保险
- **保障范围**：因意外伤害导致的身故、残疾、医疗费用。
- **适用对象**：企事业单位团体（5 人以上）。
- **保险期限**：1 年。
- **保障项目**：意外身故/残疾（10 万~100 万）、意外医疗（1 万~10 万）、住院津贴（50~200 元/天）。

### 1.4 旅行意外保险
- **保障范围**：旅行期间发生的意外伤害、医疗救援、行程延误、行李丢失等。
- **适用对象**：出境游、境内游、商务出行。
- **保险期限**：按次投保（1~30 天可选）。
- **特色服务**：24 小时全球紧急救援热线。

## 二、投保流程

```
客户咨询 → 需求分析 → 方案定制 → 报价确认 → 投保资料提交 → 核保 → 出单 → 送达保单
```

### 所需资料
- **企业客户**：营业执照复印件、投保清单、组织机构代码证。
- **个人客户**：身份证复印件、投保单签名。

## 三、理赔流程

### 一般理赔流程
```
出险报案 → 现场查勘 → 资料提交 → 定损核赔 → 赔款支付
```

### 报案方式
- **客服热线**：955XX（7×24 小时）
- **在线报案**：微信公众号「XX 保险」
- **联系客户经理**：直接联系您的专属客户经理协助报案。

### 理赔资料清单

| 险种 | 必要资料 | 补充资料 |
|------|---------|---------|
| 财产险 | 保单、出险通知书、损失清单 | 发票、维修报价单、现场照片 |
| 责任险 | 保单、索赔申请书、事故证明 | 调解书/判决书、医疗记录 |
| 意外险 | 保单、身份证、诊断证明 | 医疗发票、费用清单、伤残鉴定书 |

## 四、常见问题（FAQ）

### Q: 保单丢失了怎么办？
A: 请联系客户经理或拨打客服热线申请补发保单，补发不收取费用。

### Q: 如何查询保单状态？
A: 关注微信公众号「XX 保险」→ 我的保单，或登录官网查询。

### Q: 保险到期前会通知续保吗？
A: 我们会提前 30 天通过短信和邮件提醒续保。
