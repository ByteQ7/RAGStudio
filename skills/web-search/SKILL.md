# Web Search SKILL

## 功能
通过 SearXNG 搜索引擎在互联网上搜索信息，返回标题、链接和摘要片段。

## 使用场景
- 用户询问实时新闻或最新事件时使用
- 用户询问不熟悉的知识点时使用
- 当内置知识库中找不到相关信息时使用
- 用户要求查找参考资料或数据时使用

## 返回格式
返回的 JSON 包含以下字段：
- results[].title — 结果标题
- results[].url — 结果链接
- results[].content — 内容摘要
- results[].engine — 来源搜索引擎

## 使用建议
1. 提取用户问题中的核心关键词作为 query 参数
2. 如果搜索结果不够精确，可以换个关键词重新搜索
3. 结果中优先选择权威来源（如官方网站、学术网站）
